# Student Marks Prediction:

This project demonstrates a simple Machine Learning regression model that predicts student exam scores based on the number of hours studied. The goal is to understand how Linear Regression can model the relationship between study time and academic performance.

The dataset contains sample data of students’ study hours and corresponding marks. The model is trained using Scikit-learn's Linear Regression algorithm to learn the relationship between these variables and make predictions for unseen data.

# Technologies Used:
Python,
Pandas,
NumPy,
Scikit-learn,
Matplotlib,

# Working:
Create and load the dataset of study hours and marks.

Separate features (study hours) and target variable (marks).

Split the dataset into training and testing sets.

Train the Linear Regression model on the training data.

Evaluate the model by predicting marks on test data.

Visualize the regression line and actual data points.
